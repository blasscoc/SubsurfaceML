Data Set Information:

This dataset contains a set of daily and monthly production data as timeseries. Included is a set of oil, gas and water production, along with an ancillary suite of data including downhole pressure and temperature. This data might have use for history matching reservoir models, also included in the greater volve dataset.


Attribute Information:

2. Daily/Monthly Gas Production m^3
